# Archive

Архивные материалы проекта.

Сюда помещаются:

- старые версии макетов;
- исходные ZIP-архивы;
- выгрузки из чатов;
- устаревшие документы;
- промежуточные варианты, которые нельзя использовать как актуальные.

## Правило

Материалы из `archive/` не являются актуальными требованиями. Для актуальных требований использовать `docs/`, `project-assets/README.md` и утверждённые макеты.
