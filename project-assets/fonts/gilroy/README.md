# Gilroy Font

Папка для файлов шрифта Gilroy, если они предоставлены дизайнером или правообладателем.

## Ожидаемые файлы

```text
Gilroy-Regular.woff2
Gilroy-Medium.woff2
Gilroy-Bold.woff2
```

Допустимы также `.woff`, `.ttf`, `.otf`, если они входят в переданный комплект.

## Статус

Файлы шрифта должны быть загружены отдельно.

До подтверждения лицензии шрифт не подключать к frontend.

## Использование

Подключение Gilroy к проекту допускается только отдельной задачей после подтверждения:

1. наличие файлов;
2. наличие права коммерческого использования;
3. обновление Design System;
4. успешная проверка frontend build.
