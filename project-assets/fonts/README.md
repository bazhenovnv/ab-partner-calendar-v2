# Fonts

Папка для шрифтовых материалов проекта «АБ Афиша Бухгалтера».

## Структура

```text
project-assets/fonts/
└── gilroy/
    ├── README.md
    ├── *.woff2
    ├── *.woff
    ├── *.ttf
    └── license / notes
```

## Правило

Шрифты можно использовать в коде только после подтверждения права коммерческого использования.

На текущем этапе шрифты хранятся как дизайн-ассеты. Подключение к frontend выполняется отдельной задачей после согласования.
