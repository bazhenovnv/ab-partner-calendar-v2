# Design

Дизайн-система, UI-компоненты, правила интерфейса и эталонные визуальные решения проекта.

## Содержимое

- `design-system/` — цвета, типографика, кнопки, карточки, календарь, формы.
- `ui-states/` — hover, active, disabled, dropdown, checkbox, chevron-анимации.
- `responsive/` — правила адаптивов 1920 / 1440 / 1024 / 390.
- `notes/` — заметки по дизайну и ограничения.

## Главное правило

Design Freeze v1.0: не менять утверждённую публичную главную страницу без отдельного согласования.
