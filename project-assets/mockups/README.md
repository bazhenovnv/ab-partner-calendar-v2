# Mockups

Папка для макетов дизайнера, PDF-версий и рендеров.

## Структура

```text
mockups/
├── pdf/       # PDF-макеты: АБ Афиша main(1-4).pdf
├── renders/   # PNG/JPG рендеры PDF и страниц
├── figma/     # ссылки, экспортированные материалы, figma-json при наличии
└── old/       # старые версии макетов
```

## Основной утверждённый макет

```text
mockups/pdf/АБ Афиша main(4).pdf
```

Этот файл является эталоном для публичной главной страницы.
