# Legal Assets

Папка для юридических документов и материалов для юриста.

## Обязательные документы

- Политика конфиденциальности → `/legal/privacy`.
- Пользовательское соглашение → `/legal/terms`.
- Согласие на обработку персональных данных → `/legal/consent`.
- Политика Cookie и аналитики → `/legal/cookies`.
- Согласие на информационные рассылки → `/legal/broadcast-consent`.

## Рекомендуемые подпапки

```text
legal/
├── docx/
├── pdf/
├── lawyer-notes/
└── versions/
```

## Контакт

Во всех документах использовать только:

```text
info-event@a-b.ru
```
