# АБ Афиша Бухгалтера — дизайн-материалы для Claude

Этот архив содержит эскизы, PDF-макеты, изображения, логотип/фреймы и креативы, доступные в текущей сессии.

Ключевые файлы:

- `АБ Афиша main(4).pdf` — основной утверждённый макет для 1440;
- `АБ Афиша main(1).pdf`, `АБ Афиша main(2).pdf`, `АБ Афиша main(3).pdf` — предыдущие/смежные версии макета;
- `Frame 60.png`, `Frame 60(1).png` — фреймы/логотипные материалы;
- `render_main4/page-1.png` — рендер основного PDF-макета;
- `a_*.png` — изображения для maintenance / UI-эскизов;
- `Креативы АБ (*.zip)` — архивы креативов;
- `{GUID}.png` и `image(*).png` — загруженные скриншоты, эскизы и референсы из обсуждений.

Правило для Claude: перед любым изменением публичной главной страницы сначала открыть `АБ Афиша main(4).pdf` и сверить с Design Freeze v1.0.
