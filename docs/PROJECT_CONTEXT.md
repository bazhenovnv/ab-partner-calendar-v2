# АБ Афиша — контекст проекта

> **Статус:** черновик. Архитектура и требования не закреплены.

## Что такое АБ Афиша

АБ Афиша — публичный календарь мероприятий для бухгалтерской аудитории. Цель — дать профессионалам из сферы бухгалтерии, налогов и финансов единое место для поиска актуальных событий: конференций, вебинаров, семинаров, курсов и других профессиональных мероприятий.

## Целевая аудитория

- Бухгалтеры, финансовые директора, налоговые консультанты.
- Специалисты, которые следят за профессиональными событиями, обновлениями законодательства и обучением.

## Продукт

- Публичная афиша/каталог событий без обязательной авторизации.
- Карточки мероприятий с датой, форматом, городом, ценой, спикерами.
- Фильтрация по дате, категории, формату, городу, цене.
- Выделение важных/ключевых событий.
- Переход на регистрацию (внутреннюю или внешнюю — TBD).
- Напоминания в Telegram/MAX (TBD).
- Экспорт в календарь (TBD).

## Связь с брендом

Проект входит в экосистему АБ (АБ Партнёр). Визуальная система должна соответствовать фирменному стилю.

## Принципы разработки

Будущая разработка должна быть **строго по финальному дизайну** и одновременно **гибкой**. Без переработки архитектуры должны легко меняться:

- тексты и лейблы;
- визуальный стиль и цвета;
- блоки интерфейса (включение/отключение секций);
- список событий, категорий, фильтров, городов;
- логика интеграций (Telegram, экспорт, партнёры);
- источники данных (ручной ввод, импорт, API).
