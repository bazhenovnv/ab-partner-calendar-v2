# Проверка качества импорта MAX — 16.07.2026

После первого полного backfill:

- 98 сообщений обработано;
- 98 изображений создано;
- 40 событий опубликовано;
- 58 событий требуют внимания;
- 11 событий отмечены как главные;
- 64 связи с направлениями.

Основные найденные причины:

1. `HASHTAG_TO_DIRECTIONS` содержит только узкий набор тегов и не покрывает распространённые теги канала: бухгалтерия, отчётность, ФНС, зарплата, кадры, самозанятые и другие.
2. `getMainEvents()` смешивает активные и завершённые главные события, когда существует хотя бы одно активное.
3. Fallback карусели ограничен завершёнными `mainEvent=true`, хотя бизнес-правило требует пять последних проведённых мероприятий при отсутствии активных главных.
4. После изменения правил требуется идемпотентный повторный backfill: существующие записи обновляются по `source=MAX + externalId`, дубли не создаются.
