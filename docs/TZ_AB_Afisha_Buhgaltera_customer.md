# Техническое задание — АБ Афиша Бухгалтера

## Основные данные

- Проект: `АБ Афиша Бухгалтера`.
- Production-домен: `ab-event.pro`.
- Репозиторий: `bazhenovnv/ab-partner-calendar-v2`.
- Контактная почта на период запуска: `info-event@a-b.ru`.
- Яндекс.Метрика: `110270689`.
- VPS: Timeweb Cloud, IPv4 `77.232.136.248`, host `kvnvm-277`.

## Цель проекта

Создать сайт-календарь мероприятий для бухгалтеров с публичной страницей, админкой, фильтрами, календарём, блоком `Главные события`, цитатами, юридическими документами, аналитикой, ботами Telegram/MAX и автоматическим импортом событий из MAX.

## Ключевые требования

- Адаптивы: 1920, 1440, 1024, 390.
- Для 1440 использовать отдельный макет `АБ Афиша main(4).pdf`.
- Название блока: `Главные события`.
- События управляются через админку.
- Фильтры применяются только по кнопке `Применить`.
- Календарь показывает месячный вид и события выбранного дня.
- Кнопка `Подробнее` открывает модальное окно события.
- Кнопка `Напомнить` открывает выбор Telegram/MAX, дальше пользователь выбирает время в боте.

## Контактная почта

На период запуска используется:

```text
info-event@a-b.ru
```

Поведение клика по email на сайте:

1. Копировать email в буфер обмена.
2. Открывать почтовый клиент пользователя через `mailto:`.
3. Показывать уведомление `Email скопирован`.
4. При блокировке буфера обмена всё равно открывать почту и показывать fallback-сообщение.
