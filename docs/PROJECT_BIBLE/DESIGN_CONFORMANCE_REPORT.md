## 4. Что частично соответствует 🟡

| Компонент | Что реализовано | Что отличается |
|-----------|-----------------|----------------|
| **Hero** | Светлый фон, заголовок, subtitle, CTA | Нет декоративного изображения (книги / вазы / календарь) справа от текста, которое присутствует в PDF-макете |
| **EventCard — дата** | White box bottom-left с датой | Размер элементов даты отличается: в макете большой день (`29`-pt bold) и подпись месяца (`18`-pt bold) как отдельные строки; в коде — компактная строка `dateStr` в меньшем размере |
| **EventCard — статус «Запланировано»** | Нет бейджа | В макете detail-страницы есть mint-бейдж «Запланировано»; в карточке-листинге бейдж отсутствует — частичное расхождение |
| **RotatingQuotes — фон** | `background: #f8f9fc` (очень светло-серый) | В референсных изображениях quotes-секция может иметь более тёмный фон; конкретный цвет фона не задокументирован явно в design freeze |
| **Calendar — маркеры событий** | Цветные точки под числами (зелёная/жёлтая/серая) | В макете: полная mint-заливка ячейки при наличии события; треугольные маркеры в углу; светлый teal-фон колонок выходных. Реализация функционально корректна, но визуально упрощена |
| **Section heading при выборе даты** | Маленький `<p>` «Показаны мероприятия на {дата}» под календарём | В макете заголовок секции меняется на «События на ДД месяца ГГГГ» как H2 |
| **Footer — колонки** | 2 колонки: «Наши проекты» + «Контакты» | В maintenance-макете показаны 4 колонки: Навигация / Категории / О проекте / Документы — страницы разделов «Категории» и «О проекте» пока не реализованы |
| **Event detail — кнопки напоминания** | Отдельные кнопки: TG + MAX + ICS | В макете detail: одна кнопка «Напомнить» с иконкой колокольчика (outline) — без разделения по каналам |
